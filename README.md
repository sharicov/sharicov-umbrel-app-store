## Sharicov Umbrel Community App Store Collection

Based on a template to create an Umbrel Community App Store. These additional app stores allow developers to distribute applications without submitting to the [Official Umbrel App Store](https://github.com/getumbrel/umbrel-apps).
Goal - to have latest versions of my favorite applications

## How to use:

Add as custom app store in UmbrelOS
